# Kahyata's Consultancy Firm – Hybrid Portfolio & E-Commerce Platform

## Overview

This project is a **modern, dynamic web platform** that combines:

* Personal consultancy portfolio
* Technology services showcase
* Fully interactive e-commerce system

It is designed to represent a **real-world tech business environment**, integrating both **professional branding** and **functional product systems** in a single application.

---

## Key Features

### Portfolio & Branding

* Professional personal brand (Kahyata Muzaza)
* About section with skills and experience
* Project showcase with real-world applications

### Projects Included

* **Weed Detection AI System**
  Machine learning model for detecting weed species using image classification and object detection.

* **Accident Detection Web App (Streamlit)**
  AI-powered web application for analyzing and detecting accident scenarios from images.

---

### E-Commerce System

* Dynamic product listings
* Product filtering and categories
* Interactive product cards with animations
* Shopping cart system
* Product detail pages

---

### UI/UX & Animations

* Smooth transitions and micro-interactions
* Scroll-triggered animations
* Modern glassmorphism design
* Responsive layout (mobile-first)

---

### Smart Features

* Cart state management
* Local storage integration
* Interactive UI feedback (add-to-cart, hover effects)
* Modular and scalable structure

---

## Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Custom CSS (Glassmorphism + Modern UI)
* **Animations:** CSS animations + interactive JS

## Getting Started

1. Clone the repository:

```
git clone https://github.com/kahyatajunior/kahyata-portfolio.git
```

2. Open the project:

```
cd kahyata-portfolio
```

3. Run locally:

* Open `index.html` in your browser
  OR
* Use Live Server (recommended)
